# 🧾 MiniERP - WinForms ERP Application

---

## 🇹🇷 Türkçe Açıklama

### 📌 Proje Hakkında
MiniERP, C# ve WinForms kullanılarak geliştirilmiş masaüstü tabanlı bir ERP uygulamasıdır.  
Ürün, müşteri ve satış süreçlerini yönetmek için tasarlanmıştır.

Katmanlı mimari yapısı sayesinde sürdürülebilir ve geliştirilebilir bir yapı sunar.

---

### 🚀 Özellikler
- ✅ Ürün Yönetimi (CRUD)
- ✅ Müşteri Yönetimi (CRUD)
- ✅ Satış Yönetimi
- ✅ Satış Kalemleri (Sale Items)
- ✅ Master-Detail yapı (Satış → Satış Kalemleri)
- ✅ Katmanlı Mimari (DAL, Service, UI)
- ✅ Form tabanlı kullanıcı arayüzü

---

### 🛠 Kullanılan Teknolojiler
- C# (.NET)
- WinForms
- SQL Server
- ADO.NET

---

### 🧠 Mimari Yapı
- 📂 DAL (Data Access Layer)
- 📂 Service Layer (Business Logic)
- 📂 UI (Forms)

## 🇬🇧 English Description

### 📌 About the Project
MiniERP is a desktop-based ERP application developed using C# and WinForms.  
It is designed to manage products, customers, and sales processes.

The project follows a layered architecture for maintainability and scalability.

---

### 🚀 Features
- ✅ Product Management (CRUD)
- ✅ Customer Management (CRUD)
- ✅ Sales Management
- ✅ Sale Items (Detail per sale)
- ✅ Master-Detail structure (Sales → Sale Items)
- ✅ Layered Architecture (DAL, Service, UI)
- ✅ Form-based UI navigation

---

### 🛠 Technologies Used
- C# (.NET)
- WinForms
- SQL Server
- ADO.NET

---

### 🧠 Architecture
- 📂 DAL (Data Access Layer)
- 📂 Service Layer (Business Logic)
- 📂 UI (Forms)
